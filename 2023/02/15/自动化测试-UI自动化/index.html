Web自动化

Web自动化必备知识

  • 流程

    • 需求 -> 需求评审 -> 测试计划 -> 编写测试用例 -> 评审 -> 执行 -> 冒烟
    • 稳定功能, 正向测试流程 -> 抽取测试用例 -> 断言

  • 分层自动化

    • 理想的金字塔模型
    • 橄榄球模型, 优先进行接口测试

    Web自动化测试价值及使用场景

    • web自动化测试价值
      • 贴近用户只管
      • 提高效率: 回归测试, 功能测试加速
      • 提升质量: 兼容测试
    • web自动化测试平静
      • 维护量大
      • 误报率高
      • 覆盖率低且难以统计
    • 优化方案
      • 优先接口测试
      • 尽量选用核心功能
      • 降低脚本维护成本
      • 降低误报率
    • 适合自动化测试的项目
      • 软件需求变动不频繁
      • 项目周期长
      • 覆盖率高

网页基础 (HTML, CSS )

HTML

  • W3C标准
    • 机构 html
    • 表现 css
    • 行为 js
  • HTML框架
    • <!DOCTYPE html>用来声明当前的文档类型是 html
    • html：是网页当中最大的标签，我们称之为根标签
    • head：称之为网页的头部，它里面的内容主要用来定义网页标签及给浏览器查看的一些信息
      <meta charset="UTF-8"> ：用来定义网页的编码标准 ，国际编码
    • title：称之为网页标题标签，它里的内容会显示在浏览器的标签页上
    • body：称之为网页主体标签，它里面的内容都会显示在浏览器的白色窗口区域

  • 常见标签

    • <h1>标题</h1>

    • <p>段落</p>

    • <img src="路径" alt="名称 width = '宽度' heigh='高度'>

    • <a href="连接" > </a>

    • <div>块元素</div>

    • <span></span>

    • <br />

  • HTML表单

    • 文本输入框
    • 密码输入框
    • 单选按钮
    • 复选框
    • 下拉框
    • 提交按钮

  • 略过… 需要用的时候查询

CSS

  • 选择器: 指定要改变样式的HTML元素每条生命由一个属性和一个只组成
  • 属性: 代表设置的样式属性, 每个属性有一个值, 属性和值用冒号相连
  • CSS: 声明以分号 ; 结束, 声明以大括号括起来
  • 一般定义在<style></style>标签中

常见选择器

  • ID

    • 以#定义

    • 每个html标签id唯一

    • <p id = 'test'>Hello World!</p>

  • Class

    • 以 . 定义
    • 选择class属性
    • class可以描述一组元素

  • 元素/标签选择器
