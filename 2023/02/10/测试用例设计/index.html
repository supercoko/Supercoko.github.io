黑马测试马基础入门(黑盒测试)

测试用例

测试用例的定义

为了特定的目的而设计的一组有测试输入, 执行条件, 预期结果的案例(文档)

测试输入 -> 软件测试的核心 -> 预期结果

用例编号 项目名称 用例名称 用例属性 重要等级 预置条件 测试数据 测试步骤 预期结果
QMail-Login-001 QQ邮箱登录 填写正确的qq邮箱账号密码并成功登录 功能测试 高 QQ邮箱账号已注册成功 用户名 xxxxx@qq.com 密码: Abc123456 1. 打开qq邮箱2. 填写账号和密码 3. 点击登录 qq邮箱登录成功, 进入个人邮箱列表页面

测试用例的构成要素

  1. 用例编号 - 用例的唯一标识
  2. 用例标题 - 用例的简要描述
  3. 测试项目 - 用例所属项目范畴
  4. 用例级别 - 用例重要程度影响
  5. 预置条件 - 用例执行的前提
  6. 测试输入 - 测试用例数据输入
  7. 执行步骤 - 执行用例的步骤
  8. 预期结果 - 应该得到的结果

黑盒测试用例设计方法

等价类

  • 等价类概念

    • 在所有测试的数据中, 具有共同特征的数据子集

  • 步骤

    • 分析需求, 确定输入类型
    • 使用规则划分有效和无效等价类
    • 设计用例, 覆盖有效等价类
      • 一条用例尽可能覆盖多的有效等价类
    • 设计用例, 覆盖无效等价类
      • 一条只能覆盖一个无效等价类

  • 案例

边界值

  • 边界值定义

    • 大量的错误发生在输入输出范围的便捷上, 而不是输入范围的内部

    • 边界值: 选取正好等于, 刚刚好大于或刚刚好小于边界值作为测试数据

    • 闭区间 [0,100]
      • 上点 0, 100
      • 离点 -1, 101
      • 内点 50
    • 半开半闭区间 (0,100]
      • 上点 0, 100
      • 离点 1, 101
      • 内点 50
    • 开区间 (0,100)
      • 上点 0, 100
      • 离点 1, 99
      • 内点 50
    • 区间 0, 100
      • 上点 0, 100
      • 离点 -1, 1, 99, 101
      • 内点 50

  • 步骤

    • 分析需求, 确定输入数据类型

    • 使用规则划分有效和无效等价类

    • 确认上点, 离点, 内点

    • 设计用例, 覆盖有效等价类

      • 一条用例尽可能覆盖多的有效等价类

    • 设计用例, 覆盖无效等价类

      • 一条只能覆盖一个无效等价类

  • 案例

判定表法

  • 使用等价类方法对于输入域和输入域存在关联时无法覆盖

  • 条件桩 条件项
    动作桩 动作项

  • 步骤

    • 定义条件桩与动作桩
    • 设计优化判定表(全组合)
    • 填写动作项
    • 简化判定表
    • 抽取用例(每个规则一条用例)

  • 案例

因果图

判定表用例的数量是会根据条件数指数级增长, 此时可以采用因果法

因果图

  • 分析因和果, 考虑条件与条件之间的关系, 条件与结果之间的关系, 在转换为判定表表

  • 条件与结果之间的关系

  • 条件之间的依赖关系

  • 步骤

    • 分析需求,获取条件和动作
    • 分析条件与条件, 条件与动作之间的关系
    • 通过关系画出因果图
    • 将因果图转换为判定表

  • 案例

状态迁移法

  • 首先找到所有状态

  • 再分析各个状态之间的转换条件和转换路径

  • 从其状态迁移路径覆盖的角度来设计用例

  • 步骤

    • 明确状态节点
    • 回执状态迁移图
    • 回执装填迁移树
    • 抽取路径设计用例

  • 案例

场景法

  • 重要概念
    • 基本流
    • 备选流
    • 异常流
  • 步骤
    • 分析需求, 基本流和备选流
    • 根据基本留和备选流生成场景
    • 根据场景生成用例
  • 案例
    • 基本流 输入正确用户名, 正确验证码, 填写账户信息
    • 备选流1 用户名填写错误 报错L
    • 备选流2 输入的验证码错误 报错M

正交实验法

多用于配置测试

  • 由数理统计学科中正交试验方法金环出的一种测试多条件多输入的用例设计方法, 从大量的(实验)数据(测试例)中挑选适量的有代表性的点(例), 从而合理地暗盘实验(测试)的一种科学设计方法

  • 条件 因子 取值水平

  • 步骤

    • 分析获取因子及水平
    • 根据因子及水平数选择正交表
    • 替换因子水平, 获取实验次数
    • 细化输出测试用例
